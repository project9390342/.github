## ‚ö° Setup Guide

1. **Download** the trainer package.
2. **Extract** files into your game folder.
3. Run `SurvivorTrainer.exe` as administrator.
4. Launch Deep Rock Galactic: Survivor.
5. Press `F1` to activate the trainer overlay.

```ini
[trainer]
health=on
resources=unlimited
speed=1.25x
hotkey_health=F2
hotkey_resources=F3
```
